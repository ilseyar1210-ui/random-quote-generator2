## Описание

Random Quote Generator — это графическое приложение на Python (Tkinter) для генерации случайных вдохновляющих цитат. Приложение содержит предопределённый список цитат, позволяет добавлять свои, сохраняет историю просмотренных цитат и предоставляет фильтрацию по автору и теме.

## Установка и запуск

1. Убедитесь, что установлен Python 3.7+
2. Скачайте файл `main.py` из репозитория
3. Запустите в терминале:
   ```bash
   python main.py
